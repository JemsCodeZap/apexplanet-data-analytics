# Supermarket Sales Data Analysis

## Project Summary
This project focuses on analyzing supermarket sales data to understand sales patterns, customer behavior, and business performance.

The dataset was cleaned and Exploratory Data Analysis (EDA) was performed using Python to find useful insights and trends.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Files
- cleaned_supermarket_sales.csv
- EDA.ipynb
- scripts

## Objective
To analyze supermarket sales data and extract meaningful insights for better decision-making.
